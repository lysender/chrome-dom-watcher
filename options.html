Chrome DOM Watcher

Options

The configuration uses a JSON style format. The first level keys are variable or object names and its values are either object's elements or 1.

If the value is 1, the first level key's value is tracked rather than its elements.

About

Chrome DOM Watcher is a Google Chrome extension that allows you to set DOM objects to watch and report their values.

This simple utility is designed for developers that always need to inspect DOM objects
and watch their values for a page.
